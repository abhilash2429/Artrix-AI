Artrix AI — Widget Demo

This page demonstrates the embeddable chat widget. Click the chat button in the bottom-right corner to start a conversation.

To embed this widget on your site, add a single script tag:

<script
  src="https://cdn.yourdomain.com/widget.js"
  data-api-key="ent_live_xxxx"
  data-api-url="https://api.yourdomain.com"
  data-persona="Aria"
  data-primary-color="#0066FF"
  data-position="bottom-right"
></script>
